# Отчет по лабораторной работе №2: «Свертка изображения»

## Цель работы

Целью данной лабораторной работы является изучение и реализация различных методов свертки изображения с использованием стандартного подхода, многопоточности, многопроцессности, использования GIL и ускорения с помощью Cython.

## Структура проекта

Проект состоит из следующих файлов:

1. **`main.py`**
   Основной файл, в котором реализованы функции для свертки изображения:

   * `standard` — стандартная реализация свертки;
   * `thread` — реализация с использованием потоков;
   * `process` — реализация с использованием процессов;
   * `nogil` — оптимизация с использованием освобождения GIL;
   * `cython` — ускоренная версия на Cython.

2. **`convolution_cython.pyx`**
   Файл с реализацией функции свертки на языке Cython, предназначенной для ускорения вычислений.

3. **`setup.py`**
   Скрипт для сборки Cython-модуля, что позволяет импортировать его в Python как обычный модуль.

4. **`test.py`**
   Содержит тесты для всех функций из `main.py`, проверяет корректность работы свертки и сравнивает результаты между различными реализациями.

## Краткое описание реализации

* **Стандартная реализация (`standard`)** выполняет свертку последовательно, без параллельной обработки.
* **Потоковая реализация (`thread`)** распределяет обработку строк изображения между несколькими потоками.
* **Процессная реализация (`process`)** использует отдельные процессы для выполнения свертки, что позволяет обходить ограничения GIL.
* **Реализация с `nogil`** в Cython освобождает GIL при вычислениях, что ускоряет работу многопоточной свертки.
* **Cython-реализация (`cython`)** полностью переносит вычисления в скомпилированный модуль для достижения максимальной производительности.

## Заключение

В ходе выполнения лабораторной работы были изучены и реализованы различные подходы к свертке изображений, включая стандартную последовательную реализацию, многопоточность и многопроцессность. Были подготовлены тесты, подтверждающие корректность работы каждой реализации.

Особое внимание было уделено изучению математической основы свертки и правильной реализации алгоритма. Также был освоен подход с использованием `nogil` в Cython для ускорения вычислений, а также получены базовые навыки работы с языком Cython и его интеграцией с Python.
