# Отчет по лабораторной работе №2: «Свертка изображения»

## Цель работы

Целью данной лабораторной работы является изучение и реализация различных методов свертки изображения, включая:

* стандартный последовательный подход,
* многопоточность,
* многопроцессность,
* оптимизацию с использованием GIL в Cython,
* ускорение с помощью компиляции Cython.

Работа направлена на сравнение производительности различных методов и практическое освоение средств ускорения вычислений в Python.

## Структура проекта

Проект состоит из следующих файлов:

| Файл                         | Назначение                                                                                       |
| ---------------------------- | ------------------------------------------------------------------------------------------------ |
| **`main.py`**                | Основной файл с реализацией функций свертки: `standard`, `thread`, `process`, `nogil`, `cython`. |
| **`convolution_cython.pyx`** | Реализация функции свертки на Cython для ускорения вычислений.                                   |
| **`setup.py`**               | Скрипт сборки Cython-модуля для последующего импорта в Python.                                   |
| **`test.py`**                | Тесты для всех функций `main.py`, проверка корректности и сравнение результатов.                 |

## Краткое описание реализации

* **`standard`** — последовательная реализация свертки без параллельной обработки.
* **`thread`** — многопоточная реализация, распределяющая обработку строк изображения между потоками.
* **`process`** — многопроцессная реализация, позволяющая обойти ограничения GIL.
* **`nogil`** — Cython-реализация с освобождением GIL для ускорения многопоточной свертки.
* **`cython`** — полностью скомпилированная версия на Cython для максимальной производительности.

## Заключение 
В ходе выполнения лабораторной работы были изучены и реализованы различные подходы к свертке изображений, включая стандартную последовательную реализацию, многопоточность и многопроцессность. 

Были подготовлены тесты, подтверждающие корректность работы каждой реализации. Особое внимание было уделено изучению математической основы свертки и правильной реализации алгоритма.
